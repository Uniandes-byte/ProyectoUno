Modelos de procesamiento natural usados

Esta se encuentra en el área de aprendizaje profundo de machine learning, utiliza nodos (neuronas) en una estructura de capas. Dos librerías de Python que permiten implementar este algoritmo son Keras y Tensorflow, con las cuales se puede realizar el procesamiento del lenguaje natural y a partir de los datos suministrados (texto, o archivos) dar una respuesta al negocio, en este caso el análisis de sentimientos. A continuación, se da una descripción de la red construida.

Capa Descripción
1 Neuronas: 4
Función de activación: ReLu
2 Dropout: 0.2

3 Neuronas: 8
Función de activación: ReLu

Los resultados dependen del conjunto de datos extraídos de la vectorización de palabras siendo dependiente del número de características. Del modelo se pudo extraer el número de características positivas y negativas por clase, además que se extrajo la información de este en términos de cuánto peso le asigna a cada palabra al momento de realizar el entrenamiento. Es importante destacar que el peso es extraído de la última capa de la red neuronal (capa 5) y que las palabras se extraen directamente del objeto TfidfVectorizer().

El modelo pertenece a los modelos bayesianos, sus predicciones se realizan basadas en las probabilidades. Para este se asume una distribución multinomial, teniendo que cada uno de sus resultados son independientes y la suma de las probabilidades es igual a uno​. Este tiene los siguientes pasos:


  • Se calcula la probabilidad de cada una de las clases, en este caso si es positiva o negativa la reseña.
  • Se mira la probabilidad de que se de cada palabra dentro de la clase.
  • Para nuevos datos se evalúa si la clase es posible, calculando la probabilidad de que los nuevos datos, las palabras, pertenezcan a la clase.

En el caso de estudio se realiza una tabla en la que se tienen los atributos que se quieren predecir y en las demás columnas son las palabras, los valores de cada una es cuántas veces han aparecido para cada una:

Reseña Palabra 1 Palabra 2 Palabra ... Palabra n
Positiva #1 #3 ... #n
Negativa #2 #4 ... #n

Para el cálculo de probabilidad de cada una de las palabras en cada categoría se utiliza la siguiente ecuación:

$$ P(Palabra n|categoria) = \dfrac{\#_n}{\#_1 + \#_3 + ... + \#_n | \#_2 + \#_4 + ... + \#_n} $$

Otra operación que se realiza es calcular la probabilidad de cada categoría, teniendo así que:

$$ P(Categoria) = \dfrac{\sum{\text{Datos de las palabras de la categoria}}}{\sum{\text{Datos de todas las palabras}}}= \dfrac{\#_1 + \#_3 + ... + \#_n}{\#_1 + \#_3 + ... + \#_n | \#_2 + \#_4 + ... + \#_n} $$

Con las operaciones anteriores se calcula el producto de las probabilidades con el cual se escoge la que sea mayor:

$$ P(Categoria) \times P(\text{Palabra 1_{cat}}) \times P(\text{Palabra 2_{cat}}) \times ... \times P(\text{Palabra n_{cat}}) $$

Modelo de aprendizaje supervisado que se utiliza para la clasificación binaria, es decir, predice si una muestra pertenece a una de las dos instancias posibles. Esto se realiza mediante la relación entre características, para luego calcular la probabilidad de un resultado determinado. Para utilizar este modelo se deben realizar algunos supuestos, entre estos están


  • Solo se incluyen las variables que sean significativas, en este caso las palabras que ya se encuentran procesadas y tokenizadas.
  • Esta se debe aplicar a tamaños de muestras grandes. .
  • Las variables deben tener muy poca o ninguna multicolinealidad.
  • Se necesita que la variable de respuesta sea binaria.

El algoritmo tiene ventajas en términos de simplicidad, interpretabilidad y capacidad para modelar relaciones no lineales y la cantidad de datos. No obstante, puede tener problemas cuando se usan palabras en sentido irónico, oximoron, doble sentido o sarcásticas. Por ello entre mayor número de palabras vectorizadas el algoritmo puede mejorar, pero también a sobre ajustarse.
